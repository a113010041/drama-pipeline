# Requirement

## 專題名稱
台韓戲劇文化差異：臺灣偶像劇為何殞落？能否轉型再起？

## 目標
建立一條從素材蒐集到多平台發布的 AI 輔助內容產製流程（Pipeline），以台韓戲劇文化差異為主題，產出一篇具有分析深度的文章並發布至多個平台。

## 目標讀者
- 對台灣影視產業有興趣的一般大眾
- 關注韓流文化與亞洲戲劇市場的讀者
- 年齡層：18–35 歲

## 內容需求
- 主題：台灣 BG 偶像劇的興衰脈絡，與韓劇崛起的結構性對比
- 字數：800–1200 字
- 語言：繁體中文
- 需包含：歷史背景、結構性原因分析、OTT 市場影響、未來展望

## 平台需求
| 平台 | 格式需求 |
|------|----------|
| WordPress (cocorico.info) | 需有分類、標籤、封面圖、SEO meta |
| 第二平台（方格子 / Medium） | 純文字＋封面圖 |

## 工具清單
| 工具 | 用途 |
|------|------|
| 部落格 / 網路素材 | 素材來源 |
| ChatGPT | AI 改寫潤稿 |
| Perplexity | 事實查核與來源確認 |
| Canva | 封面圖製作 |
| WordPress | 主要發布平台 |
| 方格子 or Medium | 第二發布平台 |

## 成功標準
- 文章成功發布於 WordPress，具備完整 SEO 設定
- 文章同步發布於第二平台
- GitHub repo 有 ≥ 5 筆 commit 紀錄
- 所有文件齊全（requirement、pipeline、fact-check）
